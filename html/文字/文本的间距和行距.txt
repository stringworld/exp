字间距：
<p style=" letter-spacing:normal">

normal:默认的属性


示例 

.ls3px 
{ 
letter-spacing:3px; 
} 

.lsn3px 
{ 
letter-spacing:-3px; 
} 

正值为增大距离,负值为缩小距离. 


行间距：

line-height:3px


文字竖排
1. writing-mode（设置对象书写方向） 

语法：writing-mode : lr-tb、tb-rl 
参数：lr-tb：从左向右，从上往下 tb-rl：从上往下，从右向左 
示例：div { writing-mode: tb-rl; }
