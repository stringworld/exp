--------------------------------------------在Centos 5.5上部署apache・php・mysql--------------------------------------------------
1.先安装Centos 5.5 Server GUI版本
2.yum -y update
3.安装vmware tools[根据安装的Vmware版本号安装不同名的压缩包]
  cp VMwareTools-8.1.3-203739.tar.gz /tmp
  tar zxpf VMwareTools-8.1.3-203739.tar.gz
  cd vmware-tools-distrib
  ./vmware-install.pl
4.在Apache的Web目录下创建index.php文件
  vi /var/www/html/index.php
  <?php phpinfo();?>

5.安装mysql5.5
  rpm -Uvh http://repo.webtatic.com/yum/centos/5/latest.rpm


  

yum remove mysql mysql-*


  yum install libmysqlclient16 --enablerepo=webtatic
  yum install mysql55 mysql55-server --enablerepo=webtatic
  

service mysqld start

  mysql_upgrade


6.安装PHP5.3
  rpm -Uvh http://repo.webtatic.com/yum/centos/5/latest.rpm
  yum --enablerepo=webtatic install php
  yum --enablerepo=webtatic update php

--------------------------------------------在Centos 6.2上部署apache・php・mysql--------------------------------------------------
1.先安装Centos 6.2 Minimal版本(Customize now)选择Mysql php perl相关选项。
2.yum -y update
3.安装vmware tools[虚拟机中安装，正式服务器不需要安装]
  mkdir /mnt/cdrom
  mount /dev/cdrom /mnt/cdrom (Sysem may complain about a write only support. This is not an error, just a warning, do not take care of it)
  cd /tmp
  cp /mnt/cdrom/VMWareTools-XXX.tar.gz . (where XXX is the ref. number)
  gunzip VMWare-Tools-XXX.tar.gz
  tar xvf VMWare-Tools-XXX.tar
  cd vmware-tools-distrib
  yum install perl (if Perl is not on your system, and once again as I installed a “minimal” version of CentOS 6)
  yum -y install gcc gcc-c++ kernel-devel
  yum -y update kernel
  ./vmware-install.pl

4.在Apache的Web目录下创建index.php文件
  vi /var/www/html/index.php
  <?php phpinfo();?>

5.编写脚本，使得服务器每次重启后自动运行httpd服务
  mkdir /home/pupume/bin/apache_restart
  chmod 777 /home/pupume/bin/apache_restart
  脚本包括:
  service httpd restart
  service mysqld restart
  iptables -F
  setenforce 0
  vi /etc/rc.d/rc.local
  输入脚本:export PATH=$PATH:/home/pupume/bin/

  vi /root/.bash_profile
  输入脚本:export PATH=$PATH:/home/pupume/bin/

export PATH=$PATH:/home/skygreen/bin/
6.安装mysql5.5
  rpm -Uvh http://repo.webtatic.com/yum/centos/5/latest.rpm


  

yum remove mysql mysql-*


  yum install libmysqlclient16 --enablerepo=webtatic
  yum install mysql55 mysql55-server --enablerepo=webtatic
  

service mysqld start

  mysql_upgrade



7.安装PHP5.3
  yum remove php
  yum install php php-pear
  yum install php-mysql

--------------------------------------------修改配置--------------------------------------------------
1.修改etc/php.ini的配置
allow_call_time_pass_reference = On
register_globals = On
upload_max_filesize = 2M

2.修改文件权限:
chmod 777 templates_c
home/ele/view/default/tmp/templates_c
home/admin/view/default/tmp/templates_c

3.修改mysql的字符集为utf8
  修改etc/my.cnf
[mysqld]
character_set_server=utf8
init_connect='SET NAMES utf8'

[client]
default-character-set=utf8 

[mysql]
default-character-set=utf8 

service mysqld restart

查看字符集：
mysql：
show variables like 'character_set%' ; 

4.安装GD库
yum install php-gd
